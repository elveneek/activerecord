Готово
find($id): Находит запись по её ID.
f($id): Алиас для find($id).
__call($name, $arguments): Магический метод для обработки динамических вызовов методов.
find_by($by, $what): Находит запись по определённому полю и значению.
where(): Добавляет условие WHERE.
save(): Сохраняет текущую запись.
ioi(): Алиас для id_or_insert_id().
id_or_insert_id(): Возвращает ID или insert ID записи.
count(): Возвращает количество записей.
fast_all_of($field = 'title'): Возвращает массив значений для определённого поля (быстрая версия).
found_rows(): Возвращает количество строк, найденных запросом. По идее это count() без параметров limit, нужно чтобы построить пагинацию (и отказаться от использования более медленного  SQL_CALC_FOUND_ROWS)
paginate($per_page = 10, $current = false): Разбивает набор результатов на страницы. устанавливает Limit и Offset

НАДО ДОПИЛИТЬ
limit($limit, $count = false): Устанавливает ограничение LIMIT для SQL-запроса.
slice($pieces = 2): Разбивает набор результатов на части.

НАДО ТЕСТЫ
to_sql(): Преобразует запрос в строку SQL. Возвращает текущий sql который используется при запросе, можно использовать для отладки
to_json(): Преобразует набор результатов в строку JSON.


НАДО СДЕЛАТЬ
sql($query): Выполняет сырой произвольный SQL-запрос и создает Activerecord по нему
to_json_by_id($pretty = false): Преобразует набор результатов в строку JSON с ID в качестве ключей.
only_count(): Возвращает только количество записей (то есть SELECT count(*) в запросе, возвращает число)
size(): Возвращает размер набора результатов. Обычно совпадает с limit




truncate($are_you_sure = false): Очищает таблицу.
plus($elements = array()): Добавляет элементы в набор результатов.
by_id($id): Возвращает запись по её ID.
__construct - создание модели из массива $pages=new Page(array('data'=>$data));
paginator($activeclass = false): Генерирует пагинатор.
search(): Ищет в наборе результатов по определённым полям.
delete(): Удаляет текущую запись.
all(): Возвращает все записи.
select($select): Устанавливает предложение SELECT для SQL-запроса.
ne(): Алиас для is_not_empty().
save_and_load(): Сохраняет текущую запись и перезагружает её.
show_as($first_word, $second_word): Отображает записи в определённом формате.
